# Academic Research Skills for Claude Code

A comprehensive collection of Claude Code skills for academic research in economics, finance, and social sciences.

## Overview

This repository contains specialized skills designed to assist with various stages of the academic research workflow, from ideation to publication.

## Installation

These skills are designed for use with Claude Code CLI. To install a skill:

1. Copy the skill folder to your Claude Code skills directory
2. The skill will be available via `/skill-name` in Claude Code

## Requirements

Each skill may have specific requirements listed in its `SKILL.md` or `requirements.txt` file. Common dependencies include:
- Python 3.8+
- R with relevant packages
- Stata (for Stata-related skills)
- LaTeX distribution (for Beamer/LaTeX skills)
